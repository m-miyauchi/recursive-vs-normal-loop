# Javasript recursive vs normal-loop

## Usage

```
$ node normal_sum.js
$ node recursive_sum.js
```

## Result
```
$ node recursive_sum.js
NUM: 10000, result: 166716670000
time: 248.981ms
$ node normal_sum.js
NUM: 10000, result: 166716670000
time: 38.098ms
```
